# TeacherLEE Curriculum v2 (2026)

## 1) 트랙 구조

- **Language Track (KR/EN/JP/CN 공통 프레임)**
  - L1 Foundation (입문)
  - L2 Build (실사용)
  - L3 Exam/Career (시험·취업 특화)
- **Coding Track**
  - C1 Web Foundation (HTML/CSS)
  - C2 JavaScript Core
  - C3 Mini Project + Interview Basics

---

## 2) Language Track 상세

### L1 Foundation (4주)
- 목표
  - 기본 문장 패턴 30개 사용
  - 생존 회화 10개 상황 대응
  - 발음/억양 오류율 40%→20%
- 주차 모듈
  1. 자기소개·인사·숫자
  2. 시간·장소·이동
  3. 음식·쇼핑·요청 표현
  4. 병원/은행/긴급 상황
- 과제
  - 음성 따라하기 3회/일
  - 패턴 변형 퀴즈 10문항
  - 5문장 일기
- 완료 기준
  - 주간 퀴즈 평균 75점 이상
  - AI 회화 15세션 이상

### L2 Build (8주)
- 목표
  - 상황별 대화 확장(일상/업무)
  - 문법 정확도 80% 이상
  - 읽기 지문 500~800자 독해
- 주차 모듈
  1~2주: 시제·연결 표현  
  3~4주: 비교·가정·원인결과  
  5~6주: 이메일/메신저 실전  
  7~8주: 토론/의견 말하기
- 과제
  - 역할극 대화 2개/주
  - 문법 리라이팅 3세트/주
  - 듣기 쉐도잉 15분/일
- 완료 기준
  - 문법 테스트 80점 이상
  - 발화 길이 평균 8문장 이상

### L3 Exam/Career (6~10주)
- 목표
  - TOPIK/JLPT/HSK/TOEIC·OPIc 유형 적응
  - 이력서/면접 언어 템플릿 확보
- 모듈
  - 시험형 독해/청해/작문 루프
  - 오답 유형 분석 리포트
  - 면접 Q&A 시뮬레이션
- 완료 기준
  - 모의고사 3회 평균 목표 점수 달성
  - 약점 카테고리 2개 이하로 축소

## 3) Coding Track 상세

### C1 Web Foundation (3주)
- 목표: 시맨틱 HTML + 반응형 CSS 기초 완성
- 실습: 랜딩 페이지 1개, 폼 UI 1개
- 완료 기준: 체크리스트 80% 통과

### C2 JavaScript Core (5주)
- 목표: 변수/함수/배열/객체/비동기 기본기
- 실습: Todo, 필터링, API 호출 미니 과제
- 완료 기준: 코드 과제 10개 중 8개 통과

### C3 Project + Interview (4주)
- 목표: 미니 프로젝트 완성 + 코드 설명 능력
- 실습: 팀플형 PR 템플릿, 코드리뷰 리비전
- 완료 기준
  - 프로젝트 1개 배포
  - 면접형 질문 20개 중 16개 합격

## 4) 미션/평가 시스템

- Daily Mission 구성(권장)
  - 회화 10분
  - 문법/어휘 퀴즈 1세트
  - 코드 과제 1개(코딩 트랙 선택자)
- 점수
  - 정확도(40) + 완주율(30) + 반복학습(20) + 연속학습(10)
- 리포트
  - 주 1회: 취약점 TOP3 + 다음 주 추천 루트

## 7) 추가 권장 커리큘럼 (v2.1 확장)

### A. Language Pronunciation Clinic (발음 집중반, 2주)
- 대상: 초급~중급, 발음/억양 교정 필요 학습자
- 목표: 핵심 발음 오류 10개 교정, 문장 강세 패턴 안정화
- 과제: 1일 5문장 녹음 + AI 피드백 재녹음
- 완료 기준: 교정률 70% 이상

### B. Writing Lab (실전 작문반, 3주)
- 대상: 시험/취업 준비 학습자
- 목표: 이메일/자기소개/의견문 템플릿 완성
- 과제: 주 3회 200~400자 작문, 문장 리라이팅
- 완료 기준: 문법 정확도 85% 이상, 구조 피드백 통과

### C. Coding Pre-Project Track (프로젝트 준비반, 2주)
- 대상: C2 완료 직전 학습자
- 목표: Git/GitHub, 디버깅, 컴포넌트 분할 습관화
- 과제: 작은 기능 단위 PR 4개 작성
- 완료 기준: 체크리스트 90% 통과

### D. Portfolio Sprint (포트폴리오 완성반, 2주)
- 대상: 취업 목표 학습자
- 목표: 프로젝트 README/회고/데모 링크까지 패키징
- 과제: 배포 + 설명 영상 3분 + 기술 선택 근거 문서화
- 완료 기준: 이력서 첨부 가능한 결과물 1개 이상

### E. Interview Drill (면접 시뮬레이션, 2주)
- 대상: 시험/취업 목표 학습자
- 목표: 언어/코딩 혼합 질문 대응력 강화
- 과제: 모의 면접 6회, 답변 개선 루프
- 완료 기준: 답변 명확도·논리성·전달력 평균 80점 이상

## 8) 언어 기초 필수 커리큘럼 (신규)

> 목적: 회화/시험 이전에 반드시 필요한 **문자 체계 + 발음 + 핵심 어휘 + 문장 뼈대**를 단단히 구축

### 8-1. 공통 기초 트랙: Foundation Zero (2~4주)
- 대상: 완전 초보, 문자부터 시작해야 하는 학습자
- 학습 축
  1. 문자/기호 인식
  2. 발음 규칙
  3. 생존 어휘 300
  4. 문장 골격 20패턴
- 완료 기준
  - 문자 인식 정확도 95%+
  - 기본 발음 진단 80점+
  - 핵심 어휘 테스트 300개 중 240개 이상
  - 1~2문장 자가 소개 가능

### 8-2. 한국어 필수 기초 (KR Basics)
- 문자/발음
  - 한글 자음/모음(기본+쌍자음) 읽기
  - 받침 발음 규칙(대표 10개)
- 어휘 암기
  - 일상 명사 150, 동사/형용사 80, 필수 표현 70
- 문장 구성 핵심
  - 조사(은/는, 이/가, 을/를, 에/에서)
  - 어순(SOV), 존댓말 종결(-요) 기초
- 미션 예시
  - 자모 퀴즈 20문항/일
  - 조사 선택 퀴즈 10문항/일
  - 패턴 문장 만들기 5개/일

### 8-3. 영어 필수 기초 (EN Basics)
- 문자/발음
  - 알파벳 이름/소리, 기본 파닉스
  - 강세/리듬 기초
- 어휘 암기
  - 빈도 상위 단어 400 (A1 핵심)
- 문장 구성 핵심
  - Be동사/일반동사 기본문
  - 시제 기초(현재/과거/미래)
  - 의문문/부정문 패턴
- 미션 예시
  - 파닉스 소리 매칭 15개/일
  - 핵심 동사 문장 변환 10개/일

### 8-4. 일본어 필수 기초 (JP Basics)
- 문자/발음
  - 히라가나/가타카나 완전 정복
  - 탁음/반탁음/촉음/장음 규칙
- 어휘 암기
  - N5 필수 단어 500
- 문장 구성 핵심
  - 조사(は, が, を, に, で)
  - 기본 동사 활용(ます형/사전형 입문)
- 미션 예시
  - 문자 플래시카드 30개/일
  - 조사 채우기 10문항/일

### 8-5. 중국어 필수 기초 (CN Basics)
- 문자/발음
  - 병음(Pinyin), 4성 + 경성
  - 성모/운모 핵심 조합
- 어휘 암기
  - HSK 1~2 필수 어휘 300~400
- 문장 구성 핵심
  - 기본 어순(SVO)
  - 시간/장소/정도 부사 위치
  - 자주 쓰는 양사 20개
- 미션 예시
  - 성조 구별 퀴즈 20문항/일
  - 병음-한자 매칭 15문항/일

### 8-6. 단어 암기 시스템 (모든 언어 공통)
- 학습 방식: SRS(간격 반복)
- 1일 루틴
  - 신규 단어 20개
  - 복습 카드 60개
  - 오답 카드 재복습 15개
- 주간 목표
  - 신규 100개 + 복습 정답률 85%+
- 기능 요구사항
  - 단어장(과목/레벨별)
  - 오답 노트 자동 생성
  - 발음 듣기 + 음성 따라읽기

### 8-7. 문장 패턴 마스터 (모든 언어 공통)
- 패턴 단계
  1. 자기소개/인사
  2. 요청/허락/거절
  3. 취향/계획/경험
  4. 의견/이유/비교
- 주차별 산출물
  - 주당 패턴 8개 암기
  - 패턴 변형 문장 40개 작성
  - AI 롤플레이 5회
- 완료 기준
  - 패턴 변형 정확도 80%+
  - 롤플레이 길이 10턴 이상 유지

### 8-8. 온보딩 연동 규칙 (제품 반영)
- 온보딩에서 과목+레벨 선택 시 자동으로 아래를 먼저 할당
  - 초보: 문자/발음/기초 어휘 트랙
  - 초급: 기초 문장 패턴 + 빈출 어휘
  - 중급 이상: 실전 회화/시험 트랙 우선
- 대시보드 미션 우선순위
  1. 문자/발음 미션
  2. 단어 SRS 복습
  3. 문장 패턴 생성
  4. AI 대화 실전
