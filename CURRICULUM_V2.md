# TeacherLEE Curriculum v2 (2026)

## 1) 트랙 구조

- **Language Track (KR/EN/JP/CN 공통 프레임)**
  - L1 Foundation (입문)
  - L2 Build (실사용)
  - L3 Exam/Career (시험·취업 특화)
- **Coding Track**
  - C1 Web Foundation (HTML/CSS)
  - C2 JavaScript Core
  - C3 Mini Project + Interview Basics

---

## 2) Language Track 상세

### L1 Foundation (4주)
- 목표
  - 기본 문장 패턴 30개 사용
  - 생존 회화 10개 상황 대응
  - 발음/억양 오류율 40%→20%
- 주차 모듈
  1. 자기소개·인사·숫자
  2. 시간·장소·이동
  3. 음식·쇼핑·요청 표현
  4. 병원/은행/긴급 상황
- 과제
  - 음성 따라하기 3회/일
  - 패턴 변형 퀴즈 10문항
  - 5문장 일기
- 완료 기준
  - 주간 퀴즈 평균 75점 이상
  - AI 회화 15세션 이상

### L2 Build (8주)
- 목표
  - 상황별 대화 확장(일상/업무)
  - 문법 정확도 80% 이상
  - 읽기 지문 500~800자 독해
- 주차 모듈
  1~2주: 시제·연결 표현  
  3~4주: 비교·가정·원인결과  
  5~6주: 이메일/메신저 실전  
  7~8주: 토론/의견 말하기
- 과제
  - 역할극 대화 2개/주
  - 문법 리라이팅 3세트/주
  - 듣기 쉐도잉 15분/일
- 완료 기준
  - 문법 테스트 80점 이상
  - 발화 길이 평균 8문장 이상

### L3 Exam/Career (6~10주)
- 목표
  - TOPIK/JLPT/HSK/TOEIC·OPIc 유형 적응
  - 이력서/면접 언어 템플릿 확보
- 모듈
  - 시험형 독해/청해/작문 루프
  - 오답 유형 분석 리포트
  - 면접 Q&A 시뮬레이션
- 완료 기준
  - 모의고사 3회 평균 목표 점수 달성
  - 약점 카테고리 2개 이하로 축소

## 3) Coding Track 상세

### C1 Web Foundation (3주)
- 목표: 시맨틱 HTML + 반응형 CSS 기초 완성
- 실습: 랜딩 페이지 1개, 폼 UI 1개
- 완료 기준: 체크리스트 80% 통과

### C2 JavaScript Core (5주)
- 목표: 변수/함수/배열/객체/비동기 기본기
- 실습: Todo, 필터링, API 호출 미니 과제
- 완료 기준: 코드 과제 10개 중 8개 통과

### C3 Project + Interview (4주)
- 목표: 미니 프로젝트 완성 + 코드 설명 능력
- 실습: 팀플형 PR 템플릿, 코드리뷰 리비전
- 완료 기준
  - 프로젝트 1개 배포
  - 면접형 질문 20개 중 16개 합격

---

## 4) 미션/평가 시스템

- Daily Mission 구성(권장)
  - 회화 10분
  - 문법/어휘 퀴즈 1세트
  - 코드 과제 1개(코딩 트랙 선택자)
- 점수
  - 정확도(40) + 완주율(30) + 반복학습(20) + 연속학습(10)
- 리포트
  - 주 1회: 취약점 TOP3 + 다음 주 추천 루트

---

## 5) Free/Premium 연동 커리큘럼 정책

- Free
  - 일일 AI 채팅 20턴
  - 주간 코드 리뷰 3회
  - 모의시험 미리보기(파트 제한)
- Premium
  - 채팅/리뷰 확장
  - 개인화 리포트 무제한
  - 시험 실전팩 + 취업팩 접근

## 6) 즉시 반영할 화면 우선순위

1. `/dashboard`: 오늘의 미션을 트랙 기반으로 동적 표시
2. `/onboarding`: 목표 선택 시 트랙 추천 카드 제공
3. `/dashboard/exam`: 시험 카테고리별 약점·완료율 연결
4. `/dashboard/code`: 난이도 자동 추천 + 재도전 루프
